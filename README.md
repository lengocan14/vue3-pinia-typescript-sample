# vue3-pinia-sample

This is sample of vue3 using `pinia` state management
## What did I use in this sample
- typescript
- pinia
- written by composition api
- emit
- prop


## Commands
### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
